Used Oil LCA is an online tool for defining parametric Life Cycle Assessment (LCA) models of motor oil, performing Life Cycle Impact Assessment (LCIA), and visualizing results.

Scenario

A scenario is an LCA model composed of a top-level fragment and a list of parameters of various types. The LCA Tool contains the following two scenarios which are visible to all users.

  • Model Base Case
  • Default Scenario
