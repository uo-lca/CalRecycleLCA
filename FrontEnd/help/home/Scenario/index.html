Scenario Parameters

Scenario parameters are used to vary the base version of the fragment model. Parameters affect LCIA computation in different ways depending on type.

Parameter Types:

  • Dependency : A dependency parameter overrides the amount of fragment flow that is input to or output from a process node.
  • Process Emission : overrides a process flow emission factor.
  • Process Dissipation : overrides a process flow dissipation factor.
  • LCIA Factor : overrides a characterization factor provided by an LCIA method.

Parameter names are generated by the tool from the names of related model components.
